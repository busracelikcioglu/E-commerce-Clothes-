# E-commerce-Clothes-
patika.dev Frontend Web Development Projeleri

# Proje Hakkında

Bu proje, modern web teknolojileri kullanılarak geliştirdiğim React tabanlı bir uygulamadır. Performans, sade tasarım ve ölçeklenebilirlik ön planda tutularak tasarlanmıştır.
Uygulama geliştirme sürecinde component yapısı, yeniden kullanılabilirlik ve temiz kod prensiplerine özellikle dikkat edilmiştir.

# Kullanılan Teknolojiler
- Vite
- TypeScript
- React
- Tailwind CSS

# Projeyi Lokal Ortamda Çalıştırma

Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```
